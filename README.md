# Landing Page Bloksoftware

Proyek ini adalah landing page statis yang modern dan responsif untuk _software house_ "Bloksoftware". Dibuat murni dengan **HTML** dan **Tailwind CSS** (via CDN).

  - Headline dengan efek **gradien teks** (`from-teal-500 to-cyan-500`).
  - Sub-headline dan tombol CTA utama.
  - Layout 2 kolom (teks dan gambar) di desktop.

- **3. Features Section:**

  - Tiga kartu layanan (_card_) dengan ikon SVG, judul, deskripsi, dan efek _hover_.

- **4. About Section:**

  - Layout 2 kolom (gambar dan teks) untuk menceritakan kisah perusahaan.

- **5. Contact Form:**

  - Formulir kontak bersih dengan input untuk `Nama`, `Email`, dan `Pesan`.

- **6. Footer:**
  - Footer multi-kolom profesional dengan _branding_, tautan navigasi, layanan, legal, dan ikon media sosial.
